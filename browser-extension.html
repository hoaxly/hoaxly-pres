hoax.ly Browser Extension

The browser plugin is currently only available for Chrome. If the plugin is used accordingly, we will also create a version for Firefox.

Find it in the Chrome Webstore:

hoax.ly Browser Extension for Chrome

The browser plugin checks the current URL and matches it with the hoax.ly index. If a hit occurs, the matching review will be displayed. If a review with a negative rating has been found, the browser Extension Icon turns red.

Reviews (fact checks) for the current page/article/url will be listed and linked, so you can read it and form your own opinion.

Roadmap

Currently, usage is rather limited due to the small number of reviews in the hoaxly index, but this should change over time.

In addition, further measures are planned to provide feedback on a larger number of content:

  • Warn also if there are reviews for a visited domain.
  • Additional inclusion of sources like https://github.com/BigMcLargeHuge/opensources for problematic domains.
  • Maybe check also links in socialmedia posts.
