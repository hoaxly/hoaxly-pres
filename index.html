Die hoax.ly API

Index von Richtigstellungen als Basis

api.hoax.ly



Grundidee: Es gibt viele sehr gute Richtigstellungen, diese haben aktuell noch eher wenig Impact, vor allem im Vergleich zu den Falschmeldungen. Das wollen wir ändern. Beispiele in Deutschland: correctiv.org oder mimikama. Die API ist nichts anderes wie eine Suchmaschine, bzw. die Daten einer Suchmaschine. Wie wird sie befüllt?
